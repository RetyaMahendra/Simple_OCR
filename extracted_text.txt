It began as a simple collection of various utilities forWSGI applications and has become one of the most advanced WSGI utility libraries.Werkzeug is Unicode aware and doesn’t enforce any dependencies. It is up to the developer to choose a templateengine, database adapter, and even how to handle requests.Getting Started+ Installationo Python Version» Dependencies» Virtual environmentso Install Werkzeugo Install virtualeny+ Werkzeug Tutorial@ Introducing Shortly® Step o: A Basie WSGI Introduction@ Step 1: Creating the Folders© Step 2: The Base Structuree Intermezzo: Running the Application© Step 3: The Environment
